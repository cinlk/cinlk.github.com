简单介绍使用hexo搭建博客的方法hexo 版本为2.8.3本，本教程只适用于windows用户。

1 本地hexo博客搭建

1.1 安装git

下载msysgit安装即可。

1.2 安装Node.js

下载node.js软件进行安装即可。

1.3 安装hexo

点击鼠标右键选择git bash进入命令行终端，用npm命令安装2.8.3版本。

1.4 创建hexo项目

安装完成后在任意目录下打开git bash终端，创建hexo项目文件

1.5 初始化项目

进入创建的项目文件，执行npm初始化

生成node_modules 文件夹同时在里面会生成一些插件文件
注意：hexo是2.8.3版本需要把hexo-renderer-marked和hexo-renderer-stylus组件替换为0.1版本

1.6 运行hexo

打开浏览器输入127.0.0.1:4000，本地博客已经搭建起来了。

2 github

接下来把博客部署到github上，别人就可以看到了。

  • 先注册一个github帐号，以github的用户名建立<用户名>.github.io这样的仓库
  • 购买域名，教程参考《如何搭建一个独立博客》，域名申请后解析会花较长时间耐心等待。
  • 配置github的SSH Key

  • 在主目录里source目录下创建CNAME文件，写入绑定的域名比如test.com
